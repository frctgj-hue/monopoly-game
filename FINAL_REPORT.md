## 🚀 Как запустить

### Разработка:
```bash
# Установка зависимостей
npm install

# Запуск сервера и клиента
npm run dev

# Или отдельно:
npm run dev:server  # http://localhost:3001
npm run dev:client  # http://localhost:5173
```

### Продакшн:
```bash
npm run build
npm start
```

## 🐛 Известные ограничения

1. **Мобильная версия**: Не оптимизирована для телефонов
2. **Сохранение игр**: Игры не сохраняются при перезагрузке сервера
3. **Переподключение**: При отключении игрок выходит из игры
4. **Настройки звука**: Нет UI для управления громкостью
