## 🎭 Система анимаций

### Глобальные анимации

#### fadeIn
```css
from: opacity 0, translateY(20px)
to: opacity 1, translateY(0)
duration: 0.5s - 1s
```
Используется для: появление карточек, модальных окон, форм

#### float
```css
0%, 100%: translateY(0)
50%: translateY(-10px)
duration: 6s
```
Используется для: фоновые декоративные элементы

#### glow
```css
0%, 100%: box-shadow с opacity 0.3
50%: box-shadow с opacity 0.6
duration: 2s
```
Используется для: активные кнопки, важные элементы

#### shimmer
```css
background-position: -1000px → 1000px
duration: 2s
```
Используется для: эффект загрузки, премиум элементы
