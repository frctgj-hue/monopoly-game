# Обновление дизайна - Современный UI v4.0

Полное обновление визуального дизайна игры с переходом на современный стиль с использованием:

## 📋 Список изменений

### 1. **Tailwind Configuration** (`tailwind.config.js`)

#### Добавлены новые цвета:
```javascript
modern: {
  primary: '#6366f1',
  secondary: '#8b5cf6',
  accent: '#ec4899',
  success: '#10b981',
  warning: '#f59e0b',
  error: '#ef4444',
  dark: '#1e293b',
  light: '#f8fafc',
}
```

#### Новые шрифты:
- **Inter** - основной шрифт (современный, читаемый)
- **Poppins** - для заголовков
- **JetBrains Mono** - для моноширинного текста

#### Новые тени:
- `shadow-neo` - неоморфизм
- `shadow-neo-inset` - вдавленный неоморфизм
- `shadow-glass` - для glassmorphism
- `shadow-xl-colored` - цветные тени

#### Новые анимации:
- `animate-float` - плавающие элементы
- `animate-gradient` - анимированный градиент
- `animate-shimmer` - эффект мерцания
