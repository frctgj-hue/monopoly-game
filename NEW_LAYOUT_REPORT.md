## 📋 Выполненные изменения

### 1. Новая система уведомлений (Toast)

**App.tsx - функция showToast:**
```typescript
const showToast = (message: string, type: ToastType = 'info') => {
  const id = Date.now();
  setToasts(prev => {
    const newToasts = [...prev, { id, message, type }];
    // Ограничиваем максимум 3 сообщения
    return newToasts.slice(-3);
  });
};
```

**Что изменилось:**
- ✅ Максимум 3 сообщения одновременно
- ✅ Старые сообщения автоматически удаляются
- ✅ Новые сообщения появляются снизу
- ✅ Нет наложения друг на друга

## 🎮 Как это работает

### Для текущего игрока:
1. Карточка подсвечивается (синий градиент)
2. Показываются кубики
3. Доступны кнопки управления
4. Анимация пульсации

### Для других игроков:
1. Белый фон
2. Только информация (деньги, недвижимость)
3. Нет кнопок управления

### Для своего игрока (не в свой ход):
1. Белый фон
2. Индикатор "Ожидание..."
3. Нет активных кнопок

### Для банкротов:
1. Серый фон, полупрозрачность
2. Бейдж "Банкрот"
3. Нет кнопок
