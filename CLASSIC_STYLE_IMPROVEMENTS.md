# 🎨 Улучшение интерфейса в классическом стиле Монополии

## Дата: 2026-05-09

## Концепция

Интерфейс улучшен с сохранением классического стиля настольной игры Монополия:
- 🎯 Строгие черные границы
- 📐 Четкие прямоугольные формы
- 🎨 Классические цвета (зеленый, красный, желтый)
- 💼 Деловой стиль с заглавными буквами
- 🃏 Карточки недвижимости как в настоящей игре

---

## Улучшения PropertyManagement

### 1. Карточки недвижимости

**Стиль настоящих карточек Монополии:**

```
┌─────────────────────────┐
│ ████████████████████████ │ ← Цветная полоска (цвет группы)
│ ─────────────────────── │
│  УЛИЦА АРБАТ            │ ← Название (жирный, заглавные)
│  БЕЗ ПОСТРОЕК           │ ← Статус
│                         │
│              ┌────────┐ │
│              │ Цена   │ │ ← Цена в рамке
│              │  $60   │ │
│              └────────┘ │
│                         │
│  ┌──────────────────┐   │
│  │ 💰 ЗАЛОЖИТЬ +$30 │   │ ← Кнопки с тенью
│  └──────────────────┘   │
└─────────────────────────┘
```

### 2. Визуальные элементы

#### Границы:
- **Толщина**: `border-4` (4px) - жирные черные границы
- **Цвет**: Всегда черный (`border-black`)
- **Стиль**: Прямые углы, без скруглений (или минимальные)

#### Тени:
- **Box-shadow**: `4px 4px 0px rgba(0,0,0,0.3)` - плоская тень
- **Стиль**: Не размытая, четкая, как у настоящих карточек

#### Цвета:
- **Фон**: Кремовый `#F5F0E8` (как игровое поле)
- **Заголовок**: Зеленый `#2d8659` (классический цвет Монополии)
- **Кнопки**: Желтый `#FBBF24`, Зеленый `#10B981`

#### Типографика:
- **Шрифт**: `font-black` (900 weight)
- **Регистр**: `uppercase` - все заглавные
- **Tracking**: `tracking-wide` - увеличенный межбуквенный интервал

### 3. Кнопки

**Классический стиль с тенью:**

```tsx
<button className="
  bg-yellow-400 
  border-3 border-black 
  font-black uppercase 
  rounded
  shadow-md
  active:translate-y-0.5
" style={{
  boxShadow: '3px 3px 0px rgba(0,0,0,0.3)'
}}>
  💰 ЗАЛОЖИТЬ +$30
</button>
```

**Эффекты:**
- ✅ Плоская тень (не размытая)
- ✅ Нажатие: `active:translate-y-0.5` (кнопка "вдавливается")
- ✅ Hover: изменение цвета фона
- ✅ Disabled: серый цвет, без тени

## Технические детали

### Структура карточки:

```tsx
<div className="bg-white rounded border-4 border-black shadow-lg" 
  style={{ boxShadow: '4px 4px 0px rgba(0,0,0,0.3)' }}>
  
  {/* Цветная полоска */}
  <div className="bg-red-600 h-10 border-b-3 border-black">
    <div className="text-white font-black uppercase">
      НЕДВИЖИМОСТЬ
    </div>
  </div>

  {/* Содержимое */}
  <div className="p-3">
    <div className="font-black uppercase">УЛИЦА АРБАТ</div>
    <div className="font-bold uppercase">БЕЗ ПОСТРОЕК</div>
    
    {/* Цена */}
    <div className="bg-gray-100 border-3 border-black">
      <div className="font-black">Цена</div>
      <div className="font-black text-lg">$60</div>
    </div>

    {/* Кнопки */}
    <button className="bg-yellow-400 border-3 border-black font-black uppercase">
      💰 ЗАЛОЖИТЬ +$30
    </button>
  </div>
</div>
```
