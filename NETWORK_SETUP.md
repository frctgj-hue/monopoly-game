# Инструкция для игры по сети

## Для хоста (вы)

**Ваш IP адрес**: `192.168.0.107`

### Серверы запущены:
- Backend: `http://192.168.0.107:3001` ✅
- Frontend: `http://192.168.0.107:5173` ✅

### Что нужно сделать:

1. **Откройте игру в браузере**:
   ```
   http://192.168.0.107:5173
   ```

2. **Создайте новую игру** (кнопка "Создать игру")

3. **Скопируйте ID игры** (появится после создания)

4. **Отправьте другу**:
   - URL: `http://192.168.0.107:5173`
   - ID игры: `xxxxxx`

---

## Для друга

### Требования:
- Находиться в **той же Wi-Fi сети**, что и хост
- Современный браузер (Chrome, Firefox, Edge)

### Как подключиться:

1. **Откройте в браузере**:
   ```
   http://192.168.0.107:5173
   ```

2. **Нажмите "Присоединиться к игре"**

3. **Введите**:
   - ID игры (получите от хоста)
   - Ваше имя

4. **Нажмите "Присоединиться"**

5. **Ждите**, пока хост не запустит игру

---

## Проверка подключения

Если друг не может подключиться, проверьте:

### 1. Находитесь ли вы в одной сети?
```bash
# Друг должен выполнить команду:
ping 192.168.0.107
```
Если пинг проходит - сеть работает ✅

### 2. Брандмауэр Windows
Возможно, нужно разрешить входящие подключения:
- Откройте **Панель управления** → **Брандмауэр Windows**
- **Дополнительные параметры** → **Правила для входящих подключений**
- **Создать правило** → **Порт** → **TCP** → **3001, 5173**
- **Разрешить подключение**

### 3. Проверка портов
```bash
# На вашем компьютере:
netstat -ano | grep -E ":(3001|5173)"
```
Должны быть открыты порты 3001 и 5173 ✅

---

## Альтернатива: ngrok (если не работает локальная сеть)

Если друг не в той же сети или есть проблемы с брандмауэром:

1. **Установите ngrok**: https://ngrok.com/download

2. **Запустите туннель**:
   ```bash
   ngrok http 5173
   ```

3. **Скопируйте публичный URL** (например: `https://abc123.ngrok.io`)

4. **Отправьте другу этот URL** вместо `http://192.168.0.107:5173`

## Количество игроков

- Минимум: **2 игрока**
- Максимум: **4 игрока**

Хост может запустить игру, когда все игроки присоединились (кнопка "Начать игру").

---

## Возможные проблемы

### "Не удается подключиться к серверу"
- Проверьте, что серверы запущены
- Проверьте брандмауэр
- Убедитесь, что вы в одной сети

### "Игра не найдена"
- Проверьте правильность ID игры
- Убедитесь, что хост создал игру

### "Игра уже началась"
- Нельзя присоединиться к начатой игре
- Хост должен создать новую игру
