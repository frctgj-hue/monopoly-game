## 👤 Для вас (хост)

### 1. Откройте игру:
```
https://ripe-walls-win.loca.lt
```

⚠️ **Важно**: При первом открытии localtunnel покажет страницу с предупреждением:
- Нажмите **"Click to Continue"**
- Это нормально - localtunnel показывает это для безопасности

### 2. Создайте игру:
- Нажмите **"Создать игру"**
- Введите ваше имя
- Скопируйте **ID игры** (например: `abc123`)

### 3. Отправьте другу:
```
URL: https://ripe-walls-win.loca.lt
ID игры: [ваш ID]
```

## 🎯 Правила игры

### Количество игроков:
- Минимум: **2 игрока**
- Максимум: **4 игрока**

### Как играть:
1. **Бросайте кубики** в свой ход
2. **Покупайте недвижимость**, на которую попали
3. **Стройте дома** на полных цветовых группах
4. **Платите аренду** владельцам недвижимости
5. **Торгуйте** с другими игроками
6. **Побеждает** последний не обанкротившийся игрок

### Управление:
- **Чат**: Общайтесь с игроками (справа внизу)
- **Управление недвижимостью**: Стройте дома, продавайте
- **Торговля**: Предлагайте сделки другим игроками
- **Аукционы**: Участвуйте в торгах

## ⚠️ Важные замечания

### Стабильность туннелей:
- **localtunnel** бесплатный, но может быть нестабильным
- Если туннель упал - нужно перезапустить
- URL может измениться при перезапуске

### Производительность:
- Возможны небольшие задержки (100-300ms)
- Это нормально для бесплатных туннелей
- Игра полностью работоспособна

### Безопасность:
- Не передавайте URL посторонним
- Туннели активны только пока запущены серверы
- После завершения игры туннели можно закрыть

## 🐛 Решение проблем

### "Не удается подключиться к серверу"
1. Проверьте, что оба туннеля запущены
2. Обновите страницу (F5)
3. Очистите кэш браузера (Ctrl+Shift+Del)

### "Click to Continue" не работает
1. Попробуйте другой браузер (Chrome, Firefox)
2. Отключите блокировщики рекламы
3. Проверьте, что JavaScript включен

### "Игра не найдена"
1. Проверьте правильность ID игры
2. Убедитесь, что хост создал игру
3. Попробуйте создать новую игру

### Туннель упал
Если URL перестал работать:
1. Перезапустите туннели (см. раздел для разработчика)
2. Отправьте новые URL другу
3. Создайте новую игру

## 🔧 Для разработчика (перезапуск туннелей)

Если нужно перезапустить туннели:

```bash
# Остановить старые туннели
pkill -f localtunnel

# Запустить новые
lt --port 5173 &  # Frontend
lt --port 3001 &  # Backend

# Получить новые URL
# Обновить .env файл с новым backend URL
# Перезапустить frontend
```
