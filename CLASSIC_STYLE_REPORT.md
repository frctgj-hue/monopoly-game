**Изменения:**

### 4. Оформление клеток

**Изменения:**
- Граница клеток: 2px черная (вместо 1px серой)
- Цветная полоса: высота 12px с черной границей снизу
- Названия: капс, жирный шрифт (monopoly-title)
- Цены: шрифт Oswald, жирный
- Hover эффект: brightness(95%) вместо shadow

**Клетка недвижимости:**
```typescript
border-2 border-black bg-white
hover:brightness-95
```

**Цветная полоса:**
```typescript

```
