## 🎯 Архитектура чата

### Клиент → Сервер:
```typescript
socket.emit('send-message', { gameId, message })
```

### Сервер → Все игроки:
```typescript
io.to(gameId).emit('chat-message', { message: ChatMessage })
```

### Структура сообщения:
```typescript
{
  id: string,           // Уникальный ID
  playerId: string,     // ID отправителя
  playerName: string,   // Имя отправителя
  playerColor: string,  // Цвет фишки
  message: string,      // Текст сообщения
  timestamp: number     // Время отправки
}
```

## 🚀 Как использовать

### Строительство домов:
1. Купить все недвижимости одного цвета (монополия)
2. Нажать кнопку "🏠 Недвижимость" в карточке игрока
3. Выбрать недвижимость и нажать "Построить дом"
4. Дом появится с анимацией искр ✨

### Чат:
1. Во время игры чат появляется в правом нижнем углу
2. Клик на заголовок - развернуть/свернуть
3. Ввести сообщение и нажать Enter или кнопку ➤
4. Все игроки увидят сообщение в реальном времени
