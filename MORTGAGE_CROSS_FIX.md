## Если крестик все еще не виден

### Проверьте в консоли:
1. Есть ли лог `🔴 Заложенная недвижимость`?
   - **ДА** → Крестик рендерится, проблема в CSS
   - **НЕТ** → `property.mortgaged` не установлен

2. Есть ли лог `🟣 property-mortgaged событие получено`?
   - **ДА** → Событие получено, проверьте `mortgaged: true`
   - **НЕТ** → Событие не приходит с сервера

### Проверьте в DevTools:
1. Откройте Elements (F12 → Elements)
2. Найдите клетку заложенной недвижимости
3. Проверьте, есть ли div с классом `z-[200]`
4. Проверьте computed styles

### Если крестик есть, но не виден:
- Проверьте, не перекрывает ли его другой элемент
- Проверьте, правильно ли применяется `z-index`
- Попробуйте изменить цвет на `text-yellow-400` для теста
