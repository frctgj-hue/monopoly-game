# Монополия - Онлайн игра

Многопользовательская онлайн-версия классической игры Монополия.

## Технологии

**Frontend:**
- React + TypeScript
- Vite
- Tailwind CSS
- Socket.io-client

**Backend:**
- Node.js + Express
- TypeScript
- Socket.io
- WebSockets для реалтайм коммуникации

## Установка и запуск

### 1. Установка зависимостей

Из корневой директории проекта:

```bash
npm install
```

Это установит зависимости для всего проекта (frontend и backend).

### 2. Запуск в режиме разработки

Запустить одновременно frontend и backend:

```bash
npm run dev
```

Или запустить отдельно:

**Backend:**
```bash
npm run dev:server
```
Сервер запустится на http://localhost:3001

**Frontend:**
```bash
npm run dev:client
```
Клиент запустится на http://localhost:5173

### 3. Сборка для продакшена

```bash
npm run build
```

### 4. Запуск продакшен версии

```bash
npm start
```

## Как играть

1. Откройте http://localhost:5173 в браузере
2. Выберите "Создать игру" или "Присоединиться к игре"
3. Если создаете игру - поделитесь ID игры с друзьями
4. Дождитесь минимум 2 игроков
5. Хост начинает игру
6. Играйте по очереди:
   - Бросайте кубики
   - Покупайте недвижимость (клик по клетке)
   - Платите аренду
   - Завершайте ход

## Правила

- 2-4 игрока
- Стартовый капитал: $1500
- Проход через "Старт": +$200
- Покупка недвижимости: клик по клетке, когда вы на ней стоите
- Аренда оплачивается автоматически при попадании на чужую недвижимость

## Анимации

Игра включает профессиональные анимации для улучшения игрового опыта:

### 🎯 Движение фишек
- Плавное перемещение фишек через каждую клетку поля
- Эффект подпрыгивания при движении
- Увеличение фишки во время хода
- Визуализация полного пути от старта до финиша

### 🏠 Строительство
- Анимация появления домов с вращением
- Эффект искр ✨ при строительстве
- Трансформация 5 домов в отель 🏨
- Цветовая кодировка по группам недвижимости

### 💰 Деньги

- Зеленый цвет для получения денег
- Красный цвет для потери денег

- Показ рубашки карточки перед открытием
- Плавная анимация раскрытия
