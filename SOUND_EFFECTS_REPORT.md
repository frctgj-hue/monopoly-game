## 📋 Что сделано

### Создан менеджер звуков (SoundManager)

**Файл**: `client/src/utils/sounds.ts`

**Технология**: Web Audio API для генерации звуков в реальном времени

**Преимущества подхода:**
- Не требует загрузки аудиофайлов
- Мгновенное воспроизведение
- Малый размер бандла
- Кроссбраузерная совместимость

---

## 🎵 Реализованные звуки

### 1. **Бросок кубиков** (`playDiceRoll`)
- Короткий щелчок
- Частота: 200Hz → 100Hz
- Длительность: 100ms
- Триггер: событие `dice-rolled`

### 2. **Покупка недвижимости** (`playPropertyBought`)
- Приятный аккорд (C5, E5, G5)
- Три ноты с задержкой
- Длительность: 300ms каждая
- Триггер: событие `property-bought`

### 3. **Строительство дома** (`playHouseBuild`)
- Звук молотка (3 удара)
- Квадратная волна 150Hz
- Интервал: 150ms между ударами
- Триггер: событие `house-built`

### 4. **Оплата аренды** (`playRentPaid`)
- Грустный нисходящий тон
- Частота: 400Hz → 200Hz
- Длительность: 300ms
- Триггер: событие `rent-paid`

### 5. **Сообщение в чате** (`playChatMessage`)
- Тихий блип 800Hz
- Длительность: 50ms
- Громкость: 0.1 (тише остальных)
- Триггер: событие `chat-message` (только чужие сообщения)

### 6. **Победа** (`playVictory`)
- Триумфальная мелодия
- Восходящая гамма: C5 → E5 → G5 → C6
- Длительность: 600ms
- Триггер: событие `game-finished`

### 7. **Банкротство** (`playBankruptcy`)
- Драматический нисходящий звук
- Частота: 300Hz → 50Hz
- Длительность: 500ms
- Триггер: событие `player-bankrupt`

### 8. **Завершение хода** (`playTurnEnd`)
- Короткий сигнал 600Hz
- Длительность: 100ms
- Триггер: событие `turn-ended`

### 9. **Взятие карточки** (`playCardDrawn`)
- Восходящий свист
- Частота: 1000Hz → 1500Hz
- Длительность: 200ms
- Триггер: событие `card-drawn`

## 🔧 Технические особенности

### AudioContext управление:
```typescript
- Автоматическое возобновление при suspend
- Проверка состояния перед воспроизведением
- Поддержка включения/выключения звуков
```

### Оптимизация:
- Синглтон паттерн для менеджера
- Минимальное использование памяти
- Нет загрузки внешних файлов
- Генерация звуков "на лету"

### Громкость:
- Основные звуки: 0.2-0.3
- Чат: 0.1 (тише)
- Все звуки с плавным затуханием (exponentialRamp)
