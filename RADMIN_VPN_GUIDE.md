## ⚠️ Важные замечания

### Требования:
1. **Оба игрока** должны быть подключены к **одной сети Radmin VPN**
2. **Хост** должен запустить серверы на своем компьютере
3. **Стабильное** интернет-соединение
4. **Современный браузер** (Chrome, Firefox, Edge)

### Производительность:
- Задержка: обычно 10-50ms (очень быстро)
- Radmin VPN работает быстрее, чем публичные туннели
- Игра полностью работоспособна

### Безопасность:
- Radmin VPN создает защищенное соединение
- Только участники вашей сети могут подключиться
- IP-адрес виден только в пределах VPN-сети

## 🐛 Решение проблем

### "Не удается подключиться к серверу"
1. Проверьте, что оба подключены к одной сети Radmin VPN
2. Проверьте, что серверы запущены у хоста
3. Проверьте IP-адрес хоста в Radmin VPN (может измениться)
4. Попробуйте отключить брандмауэр Windows временно
5. Обновите страницу (F5)

### "Игра не найдена"
1. Проверьте правильность ID игры
2. Убедитесь, что хост создал игру
3. Попробуйте создать новую игру

### IP-адрес изменился
Если ваш IP в Radmin VPN изменился:

1. Узнайте новый IP:
```bash
ipconfig | grep -A 5 "IPv4"
```

2. Измените `.env` файл:
```bash
cd monopoly-game/client
# Отредактируйте .env файл
VITE_SOCKET_URL=http://[НОВЫЙ_IP]:3001
```

3. Перезапустите клиент:
```bash
npm run dev
```

### Брандмауэр блокирует подключение
Если Windows Firewall блокирует:

1. Откройте **Панель управления** → **Брандмауэр Windows**
2. Нажмите **"Разрешить приложение через брандмауэр"**
3. Найдите **Node.js** и разрешите для **частных сетей**
4. Перезапустите серверы
