## 🔧 Технические детали

### Цветовая палитра

```css
/* Основные цвета */
--bg-primary: #1c1c1e;      /* Фон страниц */
--bg-card: #252528;         /* Фон карточек */
--accent-gold: #d4af37;     /* Золотой акцент */
--accent-green: #2d5f3f;    /* Зеленый для действий */
--gradient: linear-gradient(135deg, #d4af37 0%, #f4d03f 100%);
```

### Анимации

```css
/* fadeIn - появление элементов */
@keyframes fadeIn {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* float - плавающие элементы */
@keyframes float {
  0%, 100% {
    transform: translateY(0);
  }
  50% {
    transform: translateY(-10px);
  }
}

/* glow - свечение */
@keyframes glow {
  0%, 100% {
    box-shadow: 0 0 20px rgba(212, 175, 55, 0.3);
  }
  50% {
    box-shadow: 0 0 40px rgba(212, 175, 55, 0.6);
  }
}
```

### Неоморфные тени

```css
/* Выпуклый элемент */
box-shadow: 
  8px 8px 16px rgba(0,0,0,0.4),
  -8px -8px 16px rgba(60,60,60,0.1);

/* Вдавленный элемент */
box-shadow: 
  inset 4px 4px 8px rgba(0,0,0,0.5),
  inset -4px -4px 8px rgba(60,60,60,0.1);

/* Свечение */
box-shadow: 
  0 8px 24px rgba(212, 175, 55, 0.4),
  0 0 40px rgba(212, 175, 55, 0.2);
```

### Типографика

```css
/* Заголовки */
font-weight: 300;              /* Тонкий шрифт */
letter-spacing: 0.2em - 0.3em; /* Широкие буквы */
text-transform: uppercase;      /* Заглавные буквы */

/* Основной текст */
font-weight: 300 - 400;
letter-spacing: 0.05em - 0.1em;
line-height: 1.6;
```

## 💡 Принципы дизайна

### 1. Единообразие
Все компоненты используют одинаковую цветовую палитру, стиль теней и анимации.

### 2. Премиальность
Золотые акценты (#d4af37) и неоморфизм создают ощущение роскоши.

### 3. Плавность
Все переходы и появления элементов анимированы для живого интерфейса.

### 4. Иерархия
Важные элементы выделяются размером, цветом и позицией.

### 5. Обратная связь
Hover эффекты на всех интерактивных элементах.

## 🎨 Примеры использования

### Создание карточки

```tsx
<div 
  className="rounded-2xl p-6"
  style={{
    background: '#252528',
    boxShadow: '8px 8px 16px rgba(0,0,0,0.4), -8px -8px 16px rgba(60,60,60,0.1)',
    border: '1px solid rgba(212, 175, 55, 0.1)',
  }}
>
  {/* Содержимое */}
</div>
```

### Создание кнопки

```tsx
<button
  className="px-8 py-4 rounded-xl transition-all hover:scale-105"
  style={{
    background: '#d4af37',
    color: '#1c1c1e',
    boxShadow: '0 4px 12px rgba(212, 175, 55, 0.3)',
  }}
>
  Текст кнопки
</button>
```

### Добавление анимации

```tsx
React.useEffect(() => {
  const style = document.createElement('style');
  style.textContent = `
    @keyframes fadeIn {
      from { opacity: 0; transform: translateY(20px); }
      to { opacity: 1; transform: translateY(0); }
    }
  `;
  document.head.appendChild(style);
  return () => document.head.removeChild(style);
}, []);
```
